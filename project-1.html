Project Overview

Purpose: This project was created as part of my web development course at CareerFoundry to demonstrate mastery of full stack Javascript development and enhance user engagement, expanding a movie database, and providing a personalized movie-watching experience.

Objective: The goal of the MyFlix app project was to develop a full-featured web application from scratch, showcasing my capabilities in both server-side and client-side development. This project was designed to be a cornerstone of my professional portfolio, demonstrating my expertise in integrating robust backend functionality with a seamless user interface. The aim was to not only deliver a compelling product for movie enthusiasts but also to display cutting-edge development skills to potential employers and clients.
