Chat App

The Chat App project is designed to offer a personalized messaging experience on both iOS and Android platforms. Users can customize their chat environment with background color gradients upon signing in or signing up. Once logged in, they enjoy real-time communication capabilities, including sharing selfie videos, front camera photos, and videos with cropping options. The app also supports location sharing, enhancing user interaction within chats. This project showcases advanced skills in React Native and Firestore, focusing on creating a seamless and engaging mobile app experience.

Project Overview

Purpose:The Chat App for iOS and Android is to showcase my ability to create a seamless and interactive messaging experience. This project demonstrates my skills in building robust mobile applications with a focus on user experience, utilizing React Native for app development and Google Firestore for efficient message storage and user authentication.

Objective: The objective of the Chat App project is to provide users with a customizable chat experience, enabling them to choose the background color gradient during sign-in or sign-up. Once signed in, users can communicate in real-time through various media formats, including selfie video, front video, and photos, with built-in cropping capabilities. Additionally, the app facilitates location sharing, enhancing user interactions within the chat. This project highlights my proficiency in using modern development tools and frameworks, emphasizing my expertise in React Native and Firestore.
