Meet App

This project involves building a serverless PWA with React, following TDD methodologies to ensure stability and maintainability. Key features include seamless integration with the Google Calendar API for fetching events, offline capabilities, dynamic event visualization through charts, and location-based event filtering. The app's architecture optimizes resource utilization and infrastructure costs, providing a smooth user experience across different devices and network conditions.

Project Overview

Purpose:This project is to develop a serverless, progressive web application (PWA) using React with a focus on test-driven development (TDD) principles. The application aims to integrate seamlessly with the Google Calendar API to fetch and display upcoming events, providing users with a reliable and responsive platform for scheduling and joining meetings.

Objective: The primary objective of this project is to implement a serverless architecture leveraging React to create a robust, scalable PWA. By adopting TDD practices, the development process ensures high code quality and reliability. Utilizing the Google Calendar API enhances the app's functionality, enabling users to manage events efficiently from any device.

Tools Used

React
Google Calendar API
AWS Lambda
